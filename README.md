# chatlgpd
Chat inteligente sobre a LGPD utilizando ferramentas da Microsoft Azure

Este projeto é fruto dos estudos rumo a certificação DP-100 da Microsoft a partir do bootcamp da DIO

O objetivo é criar um chatbot a partir das ferramentas da plataforma para responder questionamentos sobre a LGPD a partir de documentos previamente vetorizados.
